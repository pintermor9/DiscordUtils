Welcome to disutils's documentation!
====================================


disutils is a helper library for discord.py (or py-cord).


Features
--------

- An easy way to play music with your bot!
- Invite tracking which discord does not provide by deafult!
- And embed pagination! (v2.0 Coming soon)
