# Installation

#### Basic install
Gives access to InviteTracker and Paginator
```sh
pip install git+https://github.com/pintermor9/DiscordUtils/
```

#### Music install
Gives access to Music too.
*Requires*: discord.py[voice]
```sh
git clone https://github.com/pintermor9/DiscordUtils/
cd DiscordUtils
pip install .[voice]
```
